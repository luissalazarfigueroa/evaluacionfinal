User Journey Map

Herramienta que permite plasmar en un mapa, las etapas, interacciones, canales y elementos por los que atraviesa un cliente, y así determinar que puntos del proceso intervenir para generar una propuesta de valor

Ventajas

  1. Hace gráficas las interacciones intangibles que definen un servicio.
  2. Otorga insumos para alinear la experiencia del cliente con la visión de la organización.
  3. Permite mejorar la eficiencia operacional.
  4. Permite generar nuevas propuestas de valor

¿Cuándo utilizarlo?

Finalizada la primera planificación del servicio, como metodo de revisar la propuesta de valor.

¿Cómo se usa?

  1. Definir una "persona" o arquetipo del cliente.
  2. Determinar el recorrido del cliente y graficarlo en un eje X.
  3. Determinar las emociones respectivas del cliente en cada etapa y graficarlas en un eje Y.
  4. Analizar el gráfico obtenido y extraer puntos críticos.
  5. Proponer valor agregado para estos puntos.
