Value Proposition Canvas

Alexander Osterwalder

Plantilla de gestión estratégica para evaluar la correlación de un servicio otorgado con las necesidades del segmento apuntado.

Ventajas

  1. Permite analizar un servicio dado para prever si cumple con satisfacer las necesidades del cliente.
  2. Facilita el proceso de creación de la propuesta de valor.
  3. Es un método rápido, gráfico y de fácil entendimiento.

¿Cuándo utilizarlo?

Finalizada la primera planificación del servicio, como metodo de revisar la propuesta de valor.

¿Cómo se usa?

  1. Identificar lo que el cilente quiere resolver recurriendo al producto o servicio propuesto; Los "Trabajos".
  2. Identificar las "frustraciones" y "alegrías" del cliente.
  3. Identificar los productos o servicios que puedes construir para resolver el trabajo que tiene el cliente.
  4. Establecer como el producto o servicio ayuda al cliente.
  5. Reajustar la propuesta de valor
